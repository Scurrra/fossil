    fossil

    Fossil is a simple framework for building APIs.

    Installation

    1. Add the dependency to your shard.yml:

      dependencies:
  fossil:
    github: Scurrra/fossil

    2. Run shards install

    Usage

    require "fossil"

    A detailed guide can be found here.
